# Libraries & Dependencies

## Tech Stack Overview

GeoCam is built with modern web technologies focused on performance, accessibility, and user experience. The app leverages React 19, Vite, and TypeScript for a type-safe, fast development experience.

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Radix UI

## Core Libraries

- **react** - `^19.2.0`
  - JavaScript library for building user interfaces

- **react-dom** - `^19.2.0`
  - React rendering for web browsers

- **react-webcam** - `^7.2.0`
  - Webcam component for React - camera access

- **react-dropzone** - `^14.3.8`
  - File upload with drag and drop

- **date-fns** - `^4.1.0`
  - Modern JavaScript date utility library
